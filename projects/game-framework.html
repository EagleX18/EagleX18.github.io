Purpose

I wanted to learn how 3D rendering works with OpenGL. I had already bought a book about OpenGL, but hadn't really give it a lot of attention. A friend and I decided to start a little project for fun: making a game engine in C++ and OpenGL.

Development process

I mainly focused on the rendering side of the framework. To have a more practical course I followed some of the articles on https://learnopengl.com/ in combination with the book I had. It was a really fun project to do! Just to see how those calculations in the Fragment Shader produced those nice visuals on the screen was mindblowing.
