Full-stack web template

Purpose

I wanted to really get better in full-stack development of website applications. I often have some ideas to make applications that would require some sort of front-end administrator app to manage it. That's why I decided to start making a template that I can extend "infinitely" to make new applications. The back-end will be implemented as a RESTful API, though I'm not yet sure what language I'm going to use.

Development process

The goal is to make a front-end application that can be used with a RESTful API. This way it can easily adapt to new back-ends written in different languages. The application is written in Typescript. I like small libraries/frameworks that solve a specific problem for me, so for the data binding between "View" and "ViewModel" I decided to use KnockoutJS.
